FAQ

Les objections normales avant d'installer une app lumière.

LampControl est-il gratuit ?

Oui. L'offre gratuite permet de commencer avec les contrôles de base et jusqu'à 2 lampes.

Est-ce un abonnement ?

Non. Premium est une licence lifetime à 5,99 EUR via Lemon Squeezy.

Quelles marques de lampes connectées sont compatibles ?

LampControl prend en charge Tuya / Smart Life, Philips Hue, LIFX, Govee et Yeelight.

Comment installer LampControl sur Mac ?

Téléchargez la dernière release GitHub, ouvrez le DMG, glissez l'app dans Applications, puis lancez LampControl.

Mes clés API sont-elles envoyées ailleurs ?

Les secrets sont stockés dans le Keychain macOS. LampControl appelle les APIs nécessaires pour piloter vos fournisseurs, sans secret Lemon Squeezy embarqué.

LampControl remplace-t-il HomeKit ?

Non. LampControl est un contrôle rapide depuis la barre de menu Mac. HomeKit reste utile si votre maison connectée est déjà bien configurée.

Pourquoi utiliser LampControl plutôt que les apps Hue, LIFX ou Govee ?

LampControl évite de changer d'app pour chaque marque et met les contrôles utiles dans une seule interface macOS.

Pourquoi une app Mac plutôt qu'une app mobile ?

Parce que beaucoup de setups lumière servent le bureau: visio, focus, gaming, création, streaming. Le contrôle doit être là où vous travaillez.

Quelle version de macOS est nécessaire ?

LampControl est prévu pour macOS 13 ou plus récent.

Puis-je utiliser plusieurs marques de lampes ensemble ?

Oui. LampControl sert à regrouper plusieurs écosystèmes de lampes connectées dans une seule app Mac.

Que se passe-t-il après l'achat ?

Lemon Squeezy vous donne une clé. Collez-la dans Réglages > Premium pour activer les fonctionnalités débloquées.
